# GEBE Group

Multidisciplinary, black female-owned company delivering expert financial advisory and specialized mining services.

## Features

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Modern UI**: Built with Tailwind CSS and Framer Motion for smooth animations.
- **Modular Architecture**: Clean, component-based structure using React and TypeScript.
- **Performance**: Optimized assets and fast loading times with Vite.

## Tech Stack

- **Frontend**: React 19, TypeScript, Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Vite

## Project Structure

```text
src/
├── assets/          # Static assets (images, etc.)
├── components/      # Reusable UI components
│   ├── index.ts     # Component exports
│   └── ...          # Individual components
├── App.tsx          # Main application entry point
├── index.css        # Global styles and Tailwind configuration
└── main.tsx         # React DOM rendering
```

## Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
